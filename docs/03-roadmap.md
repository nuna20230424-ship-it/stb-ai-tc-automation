## Claude Code vs Gemini 역할 분담

### Claude Code
- 테스트 코드 자동 생성 (Pytest, Playwright 등)
- MCP 서버 작성 (STB 제어, 캡처카드, JIRA, TestRail)
- 실행 오케스트레이션 및 CI/CD 통합
- 자가치유 로직 — 실패 분석 후 코드 자동 수정

### Gemini
- 화면 캡처·동영상의 멀티모달 분석 (장기 컨텍스트 강점)
- 대용량 로그 컨텍스트 분석
- 회귀 영향도 예측, 테스트 우선순위 추천

### 라우팅 전략
- 시나리오별 강점에 따라 **A/B 비교** 후 자동 라우팅
- 비용·정확도·속도 메트릭 수집 후 모델 선택을 데이터 기반으로 결정
