# STB QA 100% AI 자동화 — 경영진 브리핑

**한 줄 요약**: 사내 Mac mini 자산을 활용해 **6개월 만에 셋탑박스 QA 100% 자동화**를 달성하는 PoC를 착수합니다. 추가 투자 **445만원**.

---

## 왜 지금인가

- **업계 표준 전환**: 2026년 글로벌 조직 77.7%가 QA에 AI 도입. Netflix·Comcast·Sky는 이미 디바이스 팜·Vision AI 기반 자동화 운영 중
- **AI 기술 성숙**: Claude Code + Gemini 멀티에이전트로 테스트 케이스 생성·실행·자가치유까지 가능 (업계 평균 **테스트 생성 시간 80%↓, 플레이키 47%↓**)
- **사내 자산 즉시 활용**: 안정성 랙의 Mac mini M4 Pro로 별도 서버 구매 없이 즉시 가동
