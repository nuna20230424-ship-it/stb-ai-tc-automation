# Disney+ 앱 통합 기능명세서 (PoC 예시)

> 2026-05-25 작성. scenario-drafter 입력 예시. 실제 KT/SKB 배포 스펙이 아닌 데모용.

## 1. 개요

STB에서 Disney+ OTT 앱을 음성 명령으로 실행하고, 4K HDR 콘텐츠를 시청·검색·자녀 보호 PIN으로 차단할 수 있어야 한다.

## 2. 진입 시나리오

### 2.1 음성으로 앱 실행
- 홈 화면에서 사용자가 리모컨 음성 버튼을 누르고 "디즈니플러스 실행"이라고 발화하면, 5초 이내에 Disney+ 홈 화면이 표시되어야 한다.
- 기대 화면: 상단 로고 + 추천 콘텐츠 row(`Trending`, `Continue Watching`).

### 2.2 IR 키로 앱 실행
- 홈 화면에서 OTT 메뉴 카드(2번째 행, 3번째 칼럼)를 OK 키로 선택 시 동일하게 진입.
- 응답 SLA 4초.

## 3. 콘텐츠 검색

### 3.1 음성 검색
- 앱 홈에서 "만달로리안 검색" 발화 시 3초 내 검색 결과 화면 표시.

### 3.2 텍스트 입력
- SEARCH 키 → on-screen 키보드 → 한 글자 입력 후 자동 추천 목록 1.5초 내 표시.

## 4. 재생 시나리오

### 4.1 4K HDR 재생 (Widevine L1)
- 앱 홈에서 첫 번째 추천 콘텐츠 카드 OK → PLAY 키 → 8초 이내 영상 표시.
- OSD에 "4K UHD HDR" 인포 표시.

### 4.2 일시정지/재개
- 재생 중 PAUSE 키 → 즉시 정지, OSD 표시 (응답 < 800ms).
- PLAY 키 → 정지 시점에서 재개.

### 4.3 빨리감기
- 재생 중 FF 키 → 2배속 OSD 표시 (응답 < 800ms).
- FF 키 추가 → 4배속.

## 5. 자녀 보호

### 5.1 PG-15 콘텐츠 차단
- 부모 PIN(0000)이 설정된 상태에서 PG-15 이상 콘텐츠 카드 OK 시 PIN 입력 다이얼로그 표시 (응답 < 3초).
- 정상 PIN 입력 후 콘텐츠 진입 가능.

### 5.2 잘못된 PIN
- 잘못된 PIN 3회 입력 시 5분 잠금 안내 메시지.

## 6. 환경

- STB 펌웨어: v2.0.0 이상
- 네트워크: 100Mbps 이상 권장
- Widevine L1 필수 (DRM 보호 콘텐츠)

## 7. 비고

- 본 명세서는 Disney+ 앱 v3.5 기준. v4.0에서 UI 개편 예정 (별도 시나리오 필요).
