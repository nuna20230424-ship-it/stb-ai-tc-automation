# STB AI Test Case Automation

셋탑박스(STB) QA 테스트케이스 100% 자동화 프로젝트.

## 목적
- STB QA 테스트케이스 작성·실행·유지보수의 100% 자동화
- 업계 벤치마킹 기반 단계별 로드맵 수립
- Claude Code / Gemini 멀티에이전트 활용 모델 정의
